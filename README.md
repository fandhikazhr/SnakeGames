# SnakeGames
Hello (ʘ‿ʘ)╯

Snake Games on your's terminal
<h3> Usage </h3>
<p> First compile file snakegames.cpp </p>
```
clang++-7 -pthread -std=c++17 -o snakegames snakegames.cpp
```
Key used :

- W for Up
- S for Down
- D for Right
- A for Left

You can play demo in here : https://replit.com/@FandhikaAzhar/snake#main.cpp
