# SnakeGames
Hello (ʘ‿ʘ)╯

Snake Games on your's terminal
<h3> Usage </h3>

First compile file `snakegames.cpp`

```
clang++ -Wall -std=c++11 snakegames.cpp -o snakegames
```

Key used :

- <kbd>W</kbd> for Up
- <kbd>S</kbd> for Down
- <kbd>D</kbd> for Right
- <kbd>A</kbd> for Left
